# Social Postify
Social Postify is an application that allows scheduling, controlling, and monitoring posts on various social media platforms.

## About

With the increase in the use of social media by companies, the creation of a tool that allows scheduling and controlling of posts has become necessary. Social Postify emerges to enable users to create and schedule posts for various social media platforms. Allowing customized posts with images, texts, and titles, the users can also select dates and times for these posts to be made. The following features are listed below:

<ul>
  <li>Add Medias</li>
  <li>Add Posts</li>
  <li>Add Publications</li>
</ul>

## How to run

1. Clone this repository
2. Install Dependencies
```bash
npm install
```

3. Run the app with
```bash
# development
npm run start

# watch mode
npm run start:dev

# production mode
npm run start:prod
```

4. Run the tests with
```bash
# unit tests
npm run test

# e2e tests
npm run test:e2e

# test coverage
npm run test:cov
```
